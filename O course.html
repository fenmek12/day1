Computer Science 101

This course provides an introduction to fundamental computer science concepts, programming, and problem-solving techniques.

Topics Covered

  1. Introduction to Computers and Computing
  2. Basics of Programming (Variables, Data Types, Operators)
  3. Control Structures (Loops, Conditionals)
  4. Functions and Modular Programming
  5. Introduction to Data Structures
